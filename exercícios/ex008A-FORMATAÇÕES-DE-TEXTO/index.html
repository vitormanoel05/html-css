Principais Formatações

Negrito (B) / Destaque (STRONG)

Nesta frase temos um termo em negrito usando a tag <b> </b> (não semântica).

Nesta frase, tenho um termo em destaque usando a tag <strong> </strong> (semântica).

Itálico (I) / Enfâse (EM)

Nesta frase temos um termo em itálico usando a tag <i> </i> (não semântica).

Nesta frase, temos um termo em enfâse usando a tag <em> </em> (semântica).

Texto Marcado (MARK)

Podemos criar também um texto marcado usando a tag <mark> </mark>.

E no outro paragráfo, temos outro texto marcado no final.

Texto Grande (BIG) / Texto Pequeno (SMALL)

Estamos criando um texto grande e um texto pequeno neste paragráfo. (a tag <big> </big> está obsoleta)

Texto Deletado (DEL)

Podemos marcar um texto como excluído para indicar que ele deve ser lida, mas não considerado.

Texto Inserido (INS) / Texto Sublinhado (U)

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o texto sublinhado.

Texto Sobrescrito (SUP)

Para inserir coisas do tipo x20+3

Texto Subscrito (SUB)

Para inserir coisas do tipo H2O
